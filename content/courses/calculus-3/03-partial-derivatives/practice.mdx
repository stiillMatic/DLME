# Chapter 3 Practice — Partial Derivatives

<ExQ>
**1.** $f = \ln(x^2 + y^2)$. Find $\nabla f$.
</ExQ>
<ExA>
$f_x = 2x/(x^2+y^2)$, $f_y = 2y/(x^2+y^2)$.

$\nabla f = 2(x, y)/(x^2+y^2)$. (Points outward, magnitude $2/\sqrt{x^2+y^2}$.)
</ExA>

<ExQ>
**2.** Directional derivative of $f = xy + z^2$ at $(1, 2, 1)$ in direction $(1, 1, 1)$.
</ExQ>
<ExA>
$\nabla f = (y, x, 2z) = (2, 1, 2)$.

$\hat{\mathbf{u}} = (1,1,1)/\sqrt 3$.

$D_{\hat{\mathbf{u}}}f = (2 + 1 + 2)/\sqrt 3 = 5/\sqrt 3 \approx 2.89$.
</ExA>

<ExQ>
**3.** Critical points of $f = x^2 - y^2 + 2x - 4y + 5$. Classify.
</ExQ>
<ExA>
$f_x = 2x + 2 = 0 \Rightarrow x = -1$. $f_y = -2y - 4 = 0 \Rightarrow y = -2$.

Hessian: $f_{xx} = 2, f_{yy} = -2, f_{xy} = 0$. $D = -4 < 0$ → **saddle** at $(-1, -2)$.
</ExA>

<ExQ>
**4. Engineering: Optimal box.** Find dimensions of rectangular box with volume 64 m³ and minimum surface area.
</ExQ>
<ExA>
$V = xyz = 64$. $S = 2(xy + yz + xz)$.

By symmetry (or Lagrange): $x = y = z = 4$ m. $S = 6\cdot 16 = 96$ m².

Cube minimizes surface for given volume.
</ExA>

<ExQ>
**5. Conceptual.** Why does Lagrange multiplier $\lambda$ measure the "cost" of the constraint?
</ExQ>
<ExA>
Mathematical fact: $d(f^*)/dc = \lambda$ where $f^*$ is the optimal value and $c$ is the constraint level.

So $\lambda$ tells you how much the optimum changes per unit relaxation of the constraint. In economics: shadow price. In engineering: marginal value.

Negative $\lambda$ means relaxing the constraint *worsens* the optimum (constraint was already actively helping).
</ExA>
