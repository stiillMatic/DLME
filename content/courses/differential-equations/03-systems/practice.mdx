# Chapter 3 Practice — Systems of ODEs

<ExQ>
**1.** Solve $\dot{\mathbf{x}} = \begin{pmatrix}2 & 0\\ 0 & -1\end{pmatrix}\mathbf{x}$ with $\mathbf{x}(0) = (1, 1)^T$.
</ExQ>
<ExA>
Decoupled: $x_1 = e^{2t}$, $x_2 = e^{-t}$. Saddle.
</ExA>

<ExQ>
**2.** Find eigenvalues of $A = \begin{pmatrix}1 & 2\\ 2 & 1\end{pmatrix}$ and classify the dynamics.
</ExQ>
<ExA>
$\det(A - \lambda I) = (1-\lambda)^2 - 4 = 0 \Rightarrow \lambda = -1, 3$.

Mixed signs: **saddle**. One direction grows ($\lambda = 3$), one decays ($\lambda = -1$).
</ExA>

<ExQ>
**3.** Solve $\dot{\mathbf{x}} = \begin{pmatrix}0 & 1\\ -1 & 0\end{pmatrix}\mathbf{x}$ with $\mathbf{x}(0) = (1, 0)^T$.
</ExQ>
<ExA>
Eigenvalues $\pm i$ — **center**. The system $\dot x_1 = x_2, \dot x_2 = -x_1$ is the rotation matrix in disguise.

$\mathbf{x}(t) = (\cos t, -\sin t)^T$.
</ExA>

<ExQ>
**4. Engineering: Two-mass coupled spring.** Two masses connected by 3 springs, $m_1 = m_2 = 1$, all spring constants $k = 1$. Equations: $\ddot x_1 = -2x_1 + x_2$, $\ddot x_2 = x_1 - 2x_2$. Find natural frequencies.
</ExQ>
<ExA>
Stiffness matrix (with mass = $I$): $K = \begin{pmatrix}2 & -1\\ -1 & 2\end{pmatrix}$. Eigenvalues = $\omega^2$.

$\det(K - \omega^2 I) = (2-\omega^2)^2 - 1 = 0 \Rightarrow \omega^2 = 1, 3$. So $\omega_1 = 1, \omega_2 = \sqrt 3$.

Mode 1: in-phase oscillation. Mode 2: out-of-phase. Standard two-mass result.
</ExA>

<ExQ>
**5. Conceptual.** Why does a $2\times 2$ system with $\det A < 0$ always have a saddle?
</ExQ>
<ExA>
$\det A = \lambda_1\lambda_2$. If $< 0$, eigenvalues have opposite signs — saddle by definition.

This is a quick stability test: just check the determinant of the linearization. Negative det → unstable in some direction.
</ExA>
