# Method of Undetermined Coefficients

## The Big Picture

When a linear ODE has a forcing term ŌĆö sinusoidal voltage, constant pressure, exponential thermal source ŌĆö we need a **particular solution** in addition to the homogeneous one. The method of **undetermined coefficients** is a smart guess: pick a trial form that matches the forcing function, then determine the coefficients by substitution. It's fast and intuitive when the forcing is simple (polynomial, exponential, sinusoidal). Combined with the homogeneous solution, it gives the full transient + steady-state response.

## Decomposition

For $L[y] = q(x)$ where $L$ is a linear differential operator:

$$y_{\text{general}} = y_h + y_p$$

- $y_h$: solution of $L[y] = 0$ (homogeneous, transient)
- $y_p$: any particular solution of $L[y] = q(x)$ (steady-state forced)

## Trial Solutions Table

| Forcing $q(x)$ | Trial $y_p$ |
|---|---|
| Polynomial of degree $n$ | Polynomial of degree $n$ with unknown coefficients |
| $e^{ax}$ | $A e^{ax}$ |
| $\sin(\omega x)$ or $\cos(\omega x)$ | $A\cos(\omega x) + B\sin(\omega x)$ |
| $x^n e^{ax}$ | $(A_n x^n + \cdots + A_0)e^{ax}$ |
| Combination | Sum of corresponding trials |

**Modification rule:** if the trial duplicates a homogeneous solution, multiply by $x$ (or $x^2$ for double resonance).

## Worked Example

Solve $y'' + y = \sin(2x)$.

**Homogeneous:** characteristic $r^2 + 1 = 0$, $r = \pm i$, $y_h = C_1\cos x + C_2\sin x$.

**Particular guess:** $y_p = A\cos(2x) + B\sin(2x)$ (since forcing isn't a homogeneous solution).

Substitute: $y_p'' = -4A\cos(2x) - 4B\sin(2x)$. So $y_p'' + y_p = -3A\cos(2x) - 3B\sin(2x)$.

Match: $-3A = 0 \Rightarrow A = 0$; $-3B = 1 \Rightarrow B = -1/3$.

$y_p = -\sin(2x)/3$. General: $y = C_1\cos x + C_2\sin x - \sin(2x)/3$.

## Resonance Case

Solve $y'' + y = \sin x$ ŌĆö same homogeneous, but forcing now matches a homogeneous solution.

**Modified guess:** $y_p = x[A\cos x + B\sin x]$.

Compute $y_p''$ via product rule, substitute. After algebra: $A = -1/2$, $B = 0$, so $y_p = -x\cos(x)/2$.

The factor $x$ shows that the response *grows linearly with time* ŌĆö **resonance**. Without damping, the oscillation amplitude grows unboundedly.

## Common Misconceptions

**Misconception 1: "Pick any guess and solve."**

The guess must structurally match the forcing. Wrong guess ŌåÆ contradictions or zero coefficients. Practice with the table.

**Misconception 2: "Resonance only matters at exactly $\omega = \omega_n$."**

In real (damped) systems, *near* resonance the amplitude is large but finite. The undamped resonance ($\omega = \omega_n$) is the limiting case where amplitude ŌåÆ Ōł×.

**Misconception 3: "$y_h$ is irrelevant if I want the steady state."**

True after transients die out. But the full IC-satisfying solution requires both. And in oscillating systems, the transient may significantly affect early-time response.

## Engineering Connection

- **Forced oscillation:** $m\ddot{x} + c\dot{x} + kx = F_0\cos(\omega t)$. Particular solution gives steady-state amplitude as a function of $\omega$ ŌĆö leading to resonance peaks in frequency response curves.
- **Driven RLC:** AC circuits at fixed frequency. Steady-state current has same frequency as voltage but with phase shift.
- **Vehicle suspension over bumps:** road profile is a periodic forcing; suspension's $y_p$ determines ride comfort.
- **Building under wind/earthquake loading:** characteristic frequency of structure must avoid forcing frequencies.

## Practice Problems

<ExQ>
Solve $y'' - 3y' + 2y = e^{3x}$.
</ExQ>
<ExA>
**Homogeneous:** $r^2 - 3r + 2 = 0 \Rightarrow r = 1, 2$. $y_h = C_1 e^x + C_2 e^{2x}$.

**Particular:** try $y_p = A e^{3x}$. Then $y_p' = 3A e^{3x}$, $y_p'' = 9A e^{3x}$.

$9A - 9A + 2A = 1 \Rightarrow 2A = 1 \Rightarrow A = 1/2$.

$y = C_1 e^x + C_2 e^{2x} + e^{3x}/2$.
</ExA>

<ExQ>
Solve $y'' + 4y = \cos(3t)$, $y(0) = 0$, $y'(0) = 0$.
</ExQ>
<ExA>
**Homogeneous:** $y_h = C_1\cos(2t) + C_2\sin(2t)$.

**Particular:** $y_p = A\cos(3t) + B\sin(3t)$. $y_p'' = -9A\cos(3t) - 9B\sin(3t)$.

$y_p'' + 4y_p = -5A\cos - 5B\sin = \cos(3t) \Rightarrow A = -1/5, B = 0$.

$y = C_1\cos(2t) + C_2\sin(2t) - \cos(3t)/5$.

ICs: $y(0) = C_1 - 1/5 = 0 \Rightarrow C_1 = 1/5$. $y'(0) = 2C_2 = 0 \Rightarrow C_2 = 0$.

$y = (\cos 2t - \cos 3t)/5$. Beat phenomenon visible.
</ExA>

<ExQ>
**Engineering: Forced spring.** A damped spring system $m\ddot x + c\dot x + kx = F_0\cos(\omega t)$ has $m = 1, c = 0.2, k = 4, F_0 = 1$. Find the steady-state amplitude when $\omega = 2$.
</ExQ>
<ExA>
Natural frequency $\omega_n = \sqrt{k/m} = 2$ ŌĆö **at resonance**!

Steady-state amplitude: $X = F_0/\sqrt{(k - m\omega^2)^2 + (c\omega)^2}$.

At resonance: $X = F_0/(c\omega) = 1/(0.2\cdot 2) = 2.5$ m.

Without damping ($c \to 0$), amplitude blows up. Damping saves the day ŌĆö but resonance amplitudes are typically much larger than off-resonance.
</ExA>

<ExQ>
**Conceptual.** When does the modification "multiply by $t$" fail (need to multiply by $t^2$ instead)?
</ExQ>
<ExA>
When the forcing matches a *double* root of the characteristic equation. E.g., $y'' - 2y' + y = e^t$ has $r = 1$ (repeated). Standard guess $y_p = Ae^t$ fails (gives 0 = 1). Guess $y_p = Ate^t$ also fails ($r = 1$ is a double root).

Multiply by another $t$: $y_p = At^2 e^t$. Now substitution gives $2A e^t = e^t \Rightarrow A = 1/2$.

The rule: multiply by $t^k$ where $k$ is the *multiplicity* of the homogeneous-root collision.
</ExA>
