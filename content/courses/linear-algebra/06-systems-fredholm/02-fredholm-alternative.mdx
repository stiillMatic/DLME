# The Fredholm Alternative Theorem

The Fredholm Alternative Theorem (FAT) is one of the most powerful results in applied mathematics. It answers the question: **when does $A\mathbf{x} = \mathbf{b}$ have a solution?**

## The Four Fundamental Subspaces

For any $m \times n$ matrix $A$, there are four fundamental subspaces:

| Subspace | Symbol | Dimension |
|---|---|---|
| Column space (range of $A$) | $\text{col}(A)$ | $r$ |
| Nullspace of $A$ | $\text{null}(A)$ | $n - r$ |
| Row space of $A$ | $\text{row}(A)$ | $r$ |
| Left nullspace of $A^T$ | $\text{null}(A^T)$ | $m - r$ |

where $r = \text{rank}(A)$. Crucially:

$$\text{col}(A) \perp \text{null}(A^T) \quad \text{and} \quad \text{row}(A) \perp \text{null}(A)$$

## Statement of the Theorem

**Exactly one of the following is true:**

**Case 1:** $A\mathbf{x} = \mathbf{b}$ has a solution $\mathbf{x}$.

**Case 2:** $A^T\mathbf{y} = \mathbf{0}$ has a solution $\mathbf{y} \neq \mathbf{0}$ with $\mathbf{y} \cdot \mathbf{b} \neq 0$.

In other words: $A\mathbf{x} = \mathbf{b}$ is solvable if and only if $\mathbf{b} \perp \text{null}(A^T)$.

## Intuition

$A\mathbf{x}$ can only produce vectors in $\text{col}(A)$. For $A\mathbf{x} = \mathbf{b}$ to be solvable, $\mathbf{b}$ must lie in $\text{col}(A)$.

Since $\text{col}(A) \perp \text{null}(A^T)$, the condition is: $\mathbf{b}$ must be orthogonal to everything in $\text{null}(A^T)$.

## The Singular Case: $\det A = 0$

When $A$ is square but singular ($\det A = 0$), the FAT tells us:

- If the solvability condition $\mathbf{b} \perp \text{null}(A^T)$ is **met**: infinitely many solutions (a family parameterized by $\text{null}(A)$)
- If the solvability condition is **not met**: no exact solution, but we can find the "best" solution via least squares

## Engineering Application: Indeterminate Structures

In structural analysis, a statically indeterminate structure yields a singular system $K\mathbf{u} = \mathbf{f}$. The FAT tells us the loading $\mathbf{f}$ must be self-equilibrated (orthogonal to rigid body modes) for a solution to exist — a displacement compatibility condition.

## Practice Problems

<ExQ>
Determine whether $A\mathbf{x} = \mathbf{b}$ is solvable for $A = \begin{pmatrix}1&2\\2&4\end{pmatrix}$, $\mathbf{b} = \begin{pmatrix}3\\6\end{pmatrix}$. If solvable, find all solutions.
</ExQ>
<ExA>
Find $\text{null}(A^T)$: $A^T = \begin{pmatrix}1&2\\2&4\end{pmatrix}$. Solve $A^T\mathbf{y}=\mathbf{0}$: $y_1 + 2y_2 = 0 \implies \mathbf{y} = t(-2,1)^T$.

Solvability: $\mathbf{y}\cdot\mathbf{b} = -2(3)+1(6) = 0$ ✓ — **solvable**.

Augmented matrix $[A|\mathbf{b}]$: row reduce $\to \begin{pmatrix}1&2&3\\0&0&0\end{pmatrix}$. Free variable $x_2 = t$.

$$\mathbf{x} = \begin{pmatrix}3\\0\end{pmatrix} + t\begin{pmatrix}-2\\1\end{pmatrix}, \quad t \in \mathbb{R}$$
</ExA>

<ExQ>
For $A = \begin{pmatrix}1&2\\2&4\end{pmatrix}$, $\mathbf{b} = \begin{pmatrix}1\\3\end{pmatrix}$: is the system solvable? Apply the Fredholm Alternative.
</ExQ>
<ExA>
From above, $\text{null}(A^T) = \text{span}\{(-2,1)^T\}$.

Solvability check: $\mathbf{y}\cdot\mathbf{b} = (-2)(1)+(1)(3) = 1 \neq 0$.

**The system is not solvable.** By the FAT, $\mathbf{b} \notin \text{col}(A)$.

The "best" (least-squares) solution minimises $\|A\mathbf{x}-\mathbf{b}\|^2$, found via $A^TA\hat{\mathbf{x}} = A^T\mathbf{b}$.
</ExA>
