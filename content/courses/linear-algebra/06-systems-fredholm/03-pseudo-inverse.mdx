# Least-Squares and the Pseudo-Inverse

When $A\mathbf{x} = \mathbf{b}$ has no solution, we look for the $\hat{\mathbf{x}}$ that minimizes the residual:

$$\min_{\mathbf{x}} \|A\mathbf{x} - \mathbf{b}\|^2$$

This is the **least-squares problem**, fundamental to data fitting, control, and machine learning.

## Normal Equations

The least-squares solution satisfies the **normal equations**:

$$A^T A\hat{\mathbf{x}} = A^T\mathbf{b}$$

If $A$ has full column rank ($\text{rank}(A) = n$), then $A^TA$ is invertible and:

$$\hat{\mathbf{x}} = (A^TA)^{-1}A^T\mathbf{b}$$

The matrix $(A^TA)^{-1}A^T$ is the **left pseudo-inverse** of $A$.

## The Moore–Penrose Pseudo-Inverse

For any matrix $A$ (including singular and rectangular), the pseudo-inverse $A^+$ is the unique matrix satisfying:

1. $AA^+A = A$
2. $A^+AA^+ = A^+$
3. $(AA^+)^T = AA^+$
4. $(A^+A)^T = A^+A$

Using SVD ($A = U\Sigma V^T$):

$$A^+ = V\Sigma^+ U^T$$

where $\Sigma^+$ replaces each non-zero singular value $\sigma_i$ with $1/\sigma_i$.

## Application: Fitting Engineering Data

Given $m$ data points $(x_i, y_i)$ and a model $y = c_0 + c_1 x + c_2 x^2$, this becomes $A\mathbf{c} = \mathbf{y}$ where:

$$A = \begin{pmatrix}1 & x_1 & x_1^2 \\ \vdots & \vdots & \vdots \\ 1 & x_m & x_m^2\end{pmatrix}$$

The least-squares solution $\hat{\mathbf{c}} = (A^TA)^{-1}A^T\mathbf{y}$ gives the best-fit polynomial.

## Application: Fitting Stress–Stretch Data

For a rubber specimen tested in tension, we measure pairs $(\lambda_i, P_i)$ (stretch, force) and fit a hyperelastic constitutive model. The pseudo-inverse provides the optimal material parameters — an example from the book connecting linear algebra to nonlinear solid mechanics.

## Practice Problems

<ExQ>
Find the least-squares fit line $y = c_0 + c_1 x$ through the points $(0, 1), (1, 2), (2, 4), (3, 4)$.
</ExQ>
<ExA>
$$A = \begin{pmatrix}1&0\\1&1\\1&2\\1&3\end{pmatrix}, \quad \mathbf{b} = \begin{pmatrix}1\\2\\4\\4\end{pmatrix}$$

$$A^TA = \begin{pmatrix}4&6\\6&14\end{pmatrix}, \quad A^T\mathbf{b} = \begin{pmatrix}11\\19\end{pmatrix}$$

Normal equations: $\begin{pmatrix}4&6\\6&14\end{pmatrix}\begin{pmatrix}c_0\\c_1\end{pmatrix} = \begin{pmatrix}11\\19\end{pmatrix}$

Solve: $c_1 = (4\cdot19 - 6\cdot11)/(4\cdot14 - 36) = (76-66)/20 = 1$, $c_0 = (11-6)/4 = 1.25$

Best-fit line: $y = 1.25 + x$
</ExA>

<ExQ>
A $2\times3$ matrix $A = \begin{pmatrix}1&0&1\\0&1&1\end{pmatrix}$ is underdetermined ($A\mathbf{x}=\mathbf{b}$ has infinitely many solutions for compatible $\mathbf{b}$). What does the pseudo-inverse $A^+$ give in this case?
</ExQ>
<ExA>
For an underdetermined system with infinitely many solutions, $A^+\mathbf{b}$ gives the **minimum-norm** solution — the unique solution with smallest $\|\mathbf{x}\|$.

This is useful in control and robotics: when a robot arm has redundant joints (more DOF than task dimensions), $A^+$ picks the joint motion requiring least energy.

For $\mathbf{b} = (1,1)^T$: the general solution is $\mathbf{x} = (1-t, 1-t, t)^T$. The minimum-norm solution is $A^+\mathbf{b} = (1/3, 1/3, 2/3)^T$ (found via SVD).
</ExA>
