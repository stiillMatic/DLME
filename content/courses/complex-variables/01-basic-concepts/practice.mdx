# Chapter 1 Practice — Basic Concepts

<ExQ>
**1.** Compute $(3 + 2i)(1 - i)$.
</ExQ>
<ExA>
$3 - 3i + 2i - 2i^2 = 3 - i + 2 = 5 - i$.
</ExA>

<ExQ>
**2.** Convert $z = 1 + i$ to polar form.
</ExQ>
<ExA>
$r = \sqrt 2$, $\theta = \pi/4$. So $z = \sqrt 2 e^{i\pi/4}$.
</ExA>

<ExQ>
**3.** Find all cube roots of 8.
</ExQ>
<ExA>
$z = 8 e^{i\cdot 0}$. Cube roots: $z_k = 2 e^{i\cdot 2\pi k/3}$ for $k = 0, 1, 2$.

$z_0 = 2$, $z_1 = -1 + i\sqrt 3$, $z_2 = -1 - i\sqrt 3$.
</ExA>

<ExQ>
**4. Conceptual.** Why does Euler's formula make trig identities easy?
</ExQ>
<ExA>
$e^{i(a+b)} = e^{ia}e^{ib}$. Expanding:

$\cos(a+b) + i\sin(a+b) = (\cos a + i\sin a)(\cos b + i\sin b)$
$= (\cos a\cos b - \sin a\sin b) + i(\sin a\cos b + \cos a\sin b)$

Real part: $\cos(a+b) = \cos a\cos b - \sin a\sin b$.
Imaginary part: $\sin(a+b) = \sin a\cos b + \cos a\sin b$.

Both classic identities — derived in one line.
</ExA>

<ExQ>
**5. Engineering.** AC current $i(t) = I_0\cos(\omega t + \phi)$. Express as the real part of a complex exponential.
</ExQ>
<ExA>
$i(t) = \text{Re}[I_0 e^{i(\omega t + \phi)}] = \text{Re}[\tilde I_0 e^{i\omega t}]$ where $\tilde I_0 = I_0 e^{i\phi}$ is the **complex amplitude (phasor)**.

Multiplication, division, derivatives all become simple algebra in phasor form. Foundation of AC circuit analysis.
</ExA>
