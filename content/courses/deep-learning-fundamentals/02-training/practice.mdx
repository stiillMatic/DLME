# Chapter 2 Practice — Training

<ExQ>
**1.** Compute MSE loss for predictions $[2.5, 1.0, 3.5]$ and targets $[2.0, 1.5, 4.0]$.
</ExQ>
<ExA>
$L = \frac{1}{3}[(0.5)^2 + (-0.5)^2 + (-0.5)^2] = \frac{1}{3}\cdot 0.75 = 0.25$.
</ExA>

<ExQ>
**2.** For loss $L = (a - y)^2$ with $a = wx + b$, find $\partial L/\partial w$.
</ExQ>
<ExA>
$\partial L/\partial a = 2(a - y)$. $\partial a/\partial w = x$.

$\partial L/\partial w = 2(a - y)x$.
</ExA>

<ExQ>
**3. Engineering: Surrogate.** A FEA surrogate has loss 25 (in stress units squared). Compute RMSE for predictions in MPa.
</ExQ>
<ExA>
RMSE = $\sqrt{25} = 5$ MPa.

For real applications: stresses ranging 0-500 MPa, RMSE of 5 MPa is ~1% — typically excellent.
</ExA>

<ExQ>
**4. Conceptual.** Why is mini-batch gradient descent preferred over full-batch?
</ExQ>
<ExA>
- Faster: don't need full dataset to compute each gradient — works with limited memory.
- Stochasticity: noise can help escape shallow minima.
- Online learning: can update with new data as it arrives.

Trade-off: smaller batch = noisier gradients; larger batch = more stable but slower per epoch. Sweet spot typically 32-256.
</ExA>

<ExQ>
**5. Multi-step.** A 1-input → 1-hidden → 1-output net with sigmoid activations. Compute one backprop step. Given $x = 1$, $w_1 = 0.5, b_1 = 0$, $w_2 = 1, b_2 = 0$, target $y = 1$, learning rate $\eta = 0.1$.
</ExQ>
<ExA>
**Forward:**
$z_1 = 0.5\cdot 1 + 0 = 0.5$. $a_1 = \sigma(0.5) = 0.622$.
$z_2 = 1\cdot 0.622 = 0.622$. $a_2 = \sigma(0.622) = 0.651$.

Loss: $L = (0.651 - 1)^2/2 = 0.0608$.

**Backward:**
$\partial L/\partial a_2 = a_2 - y = -0.349$.
$\partial L/\partial z_2 = -0.349\cdot a_2(1-a_2) = -0.349\cdot 0.227 = -0.0794$.
$\partial L/\partial w_2 = \partial L/\partial z_2 \cdot a_1 = -0.0794\cdot 0.622 = -0.0494$.

Update: $w_2 \leftarrow 1 - 0.1\cdot(-0.0494) = 1.00494$.

(Other gradients similarly. This is one SGD step.)
</ExA>
